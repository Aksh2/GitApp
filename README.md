GitHub Application to browse a list of users and view their details. The following are the technical specifications of the application:

## Technical Specifications:

- ### Architecture: MVVM
- ### UI:
- Jetpack Compose
- ### Compose Navigation
- ### DI Framework: Koin
- ### Image Lib: Coil
- ### Network Lib: Retrofit
